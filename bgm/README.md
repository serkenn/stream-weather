# BGM フォルダ

ここに**フリーBGM音源ファイル**を置いてください。配信時にアルファベット順でループ再生されます。

- 対応形式: `.mp3` `.m4a` `.aac` `.wav` `.flac` `.ogg`
- 複数置くと連続再生 → 末尾まで行ったら先頭へループ
- ファイルを差し替えても **コンテナ再ビルドは不要**（`docker compose restart` で反映）

## 注意（必ず確認）

YouTube配信では**利用許諾・著作権**を必ず確認してください。フリーでも「商用利用可」「YouTube可」「クレジット表記要否」は音源ごとに異なります。

おすすめの無料音源サイト例:
- YouTube オーディオ ライブラリ（YouTube Studio内）
- DOVA-SYNDROME / 甘茶の音楽工房 / 魔王魂 など

> このフォルダ内の音源ファイル自体は `.gitignore` で除外されます（この README のみ追跡）。
